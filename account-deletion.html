WalkStory 계정 삭제

⚠️ 주의사항
계정을 삭제하면 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다. 신중하게 결정해 주세요.

계정 삭제 시 삭제되는 데이터

다음 데이터가 영구적으로 삭제됩니다:

  • 계정 정보: 닉네임, 이메일 주소, 로그인 정보
  • 산책 기록: 모든 산책 거리, 경로, XP, 레벨
  • 게임 데이터: 수집한 동물, 장비, 인벤토리, 골드
  • 소셜 데이터: 친구 목록, 채팅 메시지, 파티 정보
  • 설정 정보: 앱 내 모든 설정 및 선호도

보관되는 데이터

다음 데이터는 법적 요구사항에 따라 일정 기간 보관될 수 있습니다:

  • 서비스 이용 기록: 법적 분쟁 해결을 위해 최대 1년간 보관
  • 결제 기록: 세법 준수를 위해 5년간 보관 (현재 결제 기능 없음)

보관 기간이 지나면 자동으로 삭제됩니다.

계정 삭제 방법

1단계: WalkStory 앱 실행
2단계: 설정 메뉴로 이동
  • 앱 하단의 프로필 아이콘 또는 헤더의 닉네임 클릭
  • 설정 메뉴 선택
3단계: 계정 삭제 요청
  • 설정 메뉴에서 "계정 삭제" 또는 "회원 탈퇴" 옵션 선택
  • 삭제 사유 선택 (선택사항)
  • 비밀번호 확인 또는 소셜 로그인 재인증
4단계: 확인 및 처리
  • 계정 삭제 확인 메시지 확인
  • 삭제 요청 제출
  • 처리 완료까지 최대 7일 소요 (즉시 처리되는 경우도 있음)

부분 데이터 삭제

계정을 삭제하지 않고 특정 데이터만 삭제하고 싶으신가요?

  • 채팅 기록 삭제: 설정 → 채팅 → 채팅 기록 삭제
  • 친구 목록 삭제: 설정 → 친구 → 친구 삭제
  • 산책 기록 삭제: 설정 → 데이터 관리 → 산책 기록 삭제

앱 내 설정 메뉴에서 개별적으로 삭제할 수 있습니다.

계정 삭제 후

계정 삭제가 완료되면:

  • 모든 데이터가 영구적으로 삭제됩니다
  • 같은 이메일/닉네임으로 재가입할 수 있습니다 (새 계정으로 시작)
  • 삭제된 데이터는 복구할 수 없습니다

문의하기

계정 삭제와 관련하여 질문이 있으시거나 도움이 필요하시면 아래로 연락해 주세요:

  • 이메일: [여기에 이메일 주소 입력]
  • 앱 내 문의: 설정 → 고객지원 → 문의하기

계정 삭제 요청도 이메일을 통해 직접 요청하실 수 있습니다.
